Description
Ce projet est une application de gestion de news développée en utilisant le framework Laravel pour le backend.

Instructions d'Installation
Clonez le dépôt : git clone https://github.com/AUARGUI-SOUKAINA/backend-app-news.git
Accédez au répertoire du projet : cd news-laravel
Installez les dépendances : composer install
Copiez le fichier .env : cp .env.example .env
Générez la clé d'application : php artisan key:generate
Configurez la base de données dans le fichier .env
Exécutez les migrations : php artisan migrate
Lancez le serveur : php artisan serve


Utilisation
L'API de news est accessible à l'adresse http://localhost:8000/api/news.
Testez l'API en utilisant Postman ou tout autre outil de votre choix.
